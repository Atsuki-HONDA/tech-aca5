# テーブル定義について

### データベース名

* board1_db

### テーブル名

* post_table
  - 投稿内容

### カラム構成


  | カラム名 | 型           | その他                | カラムの内容     |
  |----------|--------------|-----------------------|------------------|
  | id       | int          | primary key, not null | 投稿ID           |
  | name     | varchar(255) |                       | 投稿した人の名前 |
  | contents | text         |                       | 投稿された本文   |
